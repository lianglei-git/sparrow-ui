## desc-cn 
size="small" 表示小号开关。     

```html
<!-- HTML 中使用 -->
<h3>Html:</h3>
<sp-switch default-checked="true"></sp-switch>
```

```jsx

import React from 'react'
ReactDOM.render(
<div>
<h3>React:</h3>
<sp-switch default-checked="true" size='small'></sp-switch>
</div>
, mountNode);

```
