## API 
| 参数        | 说明             | 类型         | 可选 | 默认值       |
| ----------- | ---------------- | ------------ | ---- | ------------ |
| icon-show   | 展示时icon       | string\|icon | -    | sp-icon-eye  |
| icon-unshow | 不展示时icon     | string\|icon | -    | sp-icon-eyes |
| visible     | 是否显示切换按钮 | boolean      | -    | true         |

更多属性请参考 [Input](/components/input/#API)。
