## 何时使用
用来代表用户或事物，支持图片、图标或字符展示。


## API
| 参数              | 说明                             | 类型   | 可选 | 默认值 |
| ----------------- | -------------------------------- | ------ | ---- | ------ |
| shape            | 指定头像的形状                 | string | circle、square  | -      |
| size             | 设置头像的大小，支持设置具体数值 | number \| string | large、small \| number    | 40     |
| src            | 图片类头像的资源地址 | string | -    | -     |
| icon | 设置头像的图标类型，参考icon     | string | -    | -    |
